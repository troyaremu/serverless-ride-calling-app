# Serverless Ride-Calling Application — World Rydes

## Overview
A fully serverless ride-calling web application built on AWS, 
demonstrating cloud-native architecture and API-first design 
without traditional server infrastructure.

## Architecture
- **AWS Lambda** — Serverless backend functions handling 
  ride request logic
- **AWS API Gateway** — RESTful API layer managing all 
  client-server communication
- **AWS Amplify** — Frontend hosting and deployment pipeline
- **Amazon Cognito** — User authentication and access control

## Key Technical Decisions
- Fully serverless architecture eliminating server management overhead
- Event-driven design enabling automatic scaling with zero downtime
- API Gateway integration demonstrating production-grade API design
- Secure user authentication implemented via Cognito user pools
